# 5 分钟快速开始

> 从零到跑通第一个 `/sh-start`，只需 5 分钟。

---

## 前提

- 已安装 [Claude Code](https://claude.ai/code)、[Cursor](https://cursor.com)、或 [Codex](https://openai.com/codex)（任一即可）
- 有一个你想启动或推进的项目

## 第 1 步：安装 SuperpowersHarness（1 分钟）

阅读与你当前工具对应的安装说明：

- [Claude Code 安装说明](../.claude/INSTALL.md)
- [Cursor 安装说明](../.cursor/INSTALL.md)
- [Codex 安装说明](../.codex/INSTALL.md)

## 第 2 步：启动你的项目（2 分钟）

在你的 AI 工具中输入：

```text
/sh-start

- 当前项目 / 功能背景：（你的项目是什么，已有什么能力）
- 这次想解决的问题：（这次要做什么）
- 为什么现在做：（为什么是现在）
- 已知约束：（预算、时间、技术限制等）
```

不确定怎么写？看看[端到端示例](../examples/blog-ai-summary/README.md)中的输入。

写得简短也没关系——Skill 会先草拟一份模板给你确认，不会直接开跑。

## 第 3 步：拍板你的方向（1 分钟）

Skill 会给你：

1. 最多 3 条路线比较
2. 推荐路线和理由

你需要做的：**选一条路线，确认决策记录。**

## 第 4 步：进入执行（1 分钟）

方向定了之后：

```text
/sh-pack

- 当前主目标：（你刚定下的目标）
- 当前这一轮要完成的结果：（第一轮要交付什么）
- 当前明确不做：（什么留到后面）
- 依赖的上游结论 / 主文档：sh-start 拍板记录
```

拿到执行包后：

```text
/sh-run

- 当前执行包：（粘贴执行包，或指向文件路径）
- 本轮主目标：（和执行包一致）
- 本轮明确不做：（和执行包一致）
- 验收标准：（你判断"做完了"的标准）
- 哪些情况必须停下来汇报：（你不希望 AI 自作主张的边界）
```
